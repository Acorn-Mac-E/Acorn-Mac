## Overview

Acorn Professional Suite is a complete image editing platform that serves designers, developers, and content creators with the editing depth, automation capability, and format support needed for everyday professional image work without the overhead of enterprise-grade editing suites. It proves that professional-quality results do not require professional-complexity tools.

JavaScript and AppleScript automation enables batch processing, custom workflows, and integration with other applications through scripting that controls every aspect of Acorn's editing functionality programmatically. Developers use scripting to automate asset generation pipelines that produce multiple image variants from source files. The scripting bridge makes Acorn a programmable image processing engine.

Web export optimization produces compressed PNG, JPEG, WebP, and HEIC files with visual quality comparison and file size preview, enabling informed quality-versus-size decisions. The export dialog shows the visual impact of compression settings before saving, eliminating trial-and-error. Batch export applies the same optimization settings across multiple files.

Screenshot and annotation tools capture screen content and add arrows, circles, text labels, and numbered steps for documentation, bug reports, and instructional materials. Redaction tools securely obscure sensitive information in screenshots before sharing. The annotation tools are accessible from the toolbar without mode switching.

Acorn Professional Suite is the image editor for users who want professional results from a tool that respects the macOS platform, runs efficiently, and costs a fair price, without the subscription model and feature bloat that characterize the alternatives.

## Performance & Integration

Acorn is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Acorn processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
